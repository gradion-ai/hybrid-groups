This setup guide will walk you through the process of creating and configuring the Hybrid Groups Slack App in your Slack workspace.

You'll need a Slack account and permission to install apps (most users have this by default).

To create the Hybrid Groups Slack App in your workspace, you'll need an App Configuration Token. Follow these steps:

1. Navigate to Slack Apps

Go to api.slack.com/apps and log in to your Slack account. Find the section Your App Configuration Tokens.

2. Generate your token

Click Generate Token to create a new token pair, then copy the generated Access Token and paste it below.

This token is only used once for app creation and is not stored.

Slack App Created Successfully!

App Name: Hybrid Groups

To receive events from Slack in the Hybrid Groups backend, you'll need an App-Level Token. Follow these steps:

1. Go to Slack App settings

Go to the App-Level Tokens section of your newly created Slack app and click Generate Token and Scopes.

2. Generate your token

Give the token a name (e.g. App Token) and add the connections:write scope. Click Generate to create the token.

3. Copy your token

Copy the generated token and paste it below.

To use the Slack App in your workspace, it must be installed. This process generates a Bot User OAuth Token that allows the app to perform actions in your Slack workspace. Follow these steps:

1. Install your Slack App

Go to the Install App section of your Slack app and click Install to [your workspace].

2. Grant permissions

A permissions dialog will appear requesting workspace access. Review the permissions and click Allow to authorize the app.

3. Copy your Bot User OAuth Token

After installation, the OAuth Tokens section displays your Bot User OAuth Token. Copy it and paste it below.

Setup Complete!

Your Slack app has been created and configured successfully. Slack App tokens and configuration settings have been saved to the application's .env file.
