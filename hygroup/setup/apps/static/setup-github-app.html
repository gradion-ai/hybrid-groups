This setup guide will walk you through the process of creating and configuring the Hybrid Groups GitHub App in your GitHub account or organization.

You need a GitHub account to continue. If creating for an organization, you need owner permissions.

To create the Hybrid Groups GitHub App, we'll use the GitHub App Manifest Flow. This will automatically configure your app with the required permissions and webhook settings.

1. App Name

Choose a unique name for your GitHub App. This will be visible in GitHub's UI.

2. GitHub Organization (Optional)

If you want to create the app for an organization, enter its name. Leave empty to create for your personal account.

3. Webhook URL

The URL where GitHub will send webhook events. For local development, you can use smee.io to create a webhook proxy.

Leave empty to create the app for your personal account

For testing, you can use smee.io to create a webhook proxy that forwards GitHub events to your local development server.

1. Install your GitHub App

Click the button below to open GitHub and install your app. Then click Install to add it to your account or organization.

2. Grant permissions

Review the permissions requested by the app and select which repositories it can access. Click Install to proceed.

3. Copy the Installation ID

After installation, the Installation ID will be visible in the URL. Copy the number after /installations/ and paste it below.

This ID will be stored in your configuration.
